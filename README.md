# 🌐 JavaScript Validation Demo

This project demonstrates basic client-side JavaScript functionalities including:

* ✅ Email validation using Regular Expressions
* 🔐 Password strength checking
* 🍪 Handling browser cookies
* 🗂️ Using session storage

---

## 📌 Features

### 📧 Email Validation

* Validates user input using regex pattern
* Displays whether the email is valid or invalid

### 🔐 Password Strength Checker

* Checks password strength based on:

  * Length
  * Uppercase letters
  * Numbers
* Displays strength as:

  * Weak ❌
  * Medium ⚠️
  * Strong 💪

### 🍪 Cookies Handling

* Set cookies in the browser
* Retrieve and display stored cookies

### 🗂️ Session Storage

* Store data temporarily using sessionStorage
* Retrieve stored session data
